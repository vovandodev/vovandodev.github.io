Read-only user in PostgreSQL

Create only read only with limited access to PostgreSQL Database. This user can only SELECT table data and also restricted by schema access. 

1.

Script to Create Read-Only user:

  <p>
    <code>ALTER USER readuser&nbsp;</code><code>set</code>&nbsp;<code>default_transaction_read_only = on;</code>
  </p>
</td>

2.

Assign permission to this read only user:

  <p>
    <code>\c overhaul-backend_qa</code>
  </p>
  
  <p>
    <code>GRANT USAGE ON SCHEMA public TO readuser;</code>
  </p>
  
  <p>
    <code>GRANT SELECT ON ALL TABLES IN SCHEMA public TO readuser;</code>
  </p>
  
  <p>
    <code>GRANT SELECT ON ALL SEQUENCES IN SCHEMA public TO readuser;</code>
  </p>
</td>

 

3. In the above script, you can find that we have restricted user by Database, Schema and SELECT permission to tables.
